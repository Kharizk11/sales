# نظام إدارة المبيعات - فروع الشركة

تطبيق ويب لإدارة المبيعات والفروع مع دعم Firebase و LocalStorage.

## البنية التنظيمية

```
├── index.html              # الصفحة الرئيسية مع Router
├── pages/                  # صفحات HTML منفصلة
│   ├── home.html          # صفحة الرئيسية
│   ├── branches.html       # صفحة الفروع
│   ├── sales.html          # صفحة المبيعات
│   └── reports.html       # صفحة التقارير
├── js/
│   ├── core/              # الملفات الأساسية المشتركة
│   │   ├── config.js      # إعدادات Firebase والتطبيق
│   │   ├── utils.js       # الوظائف المساعدة
│   │   ├── data.js        # إدارة البيانات (Firebase/LocalStorage)
│   │   └── router.js      # نظام التنقل بين الصفحات
│   └── pages/             # ملفات JavaScript لكل صفحة
│       ├── home.js         # منطق صفحة الرئيسية
│       ├── branches.js     # منطق صفحة الفروع
│       ├── sales.js         # منطق صفحة المبيعات
│       └── reports.js      # منطق صفحة التقارير
└── css/
    └── styles.css         # ملف CSS الرئيسي
```

## المميزات

- ✅ فصل كامل للصفحات (كل صفحة في ملف HTML منفصل)
- ✅ فصل منطق JavaScript (كل صفحة لها ملف JS خاص)
- ✅ ملفات أساسية مشتركة (config, utils, data, router)
- ✅ نظام Router للتنقل بين الصفحات
- ✅ دعم Firebase و LocalStorage
- ✅ تصميم متجاوب وحديث

## كيفية الاستخدام

1. افتح `index.html` في المتصفح
2. سيتم تحميل الصفحة الرئيسية تلقائياً
3. استخدم القائمة العلوية للتنقل بين الصفحات

## الصيانة

- لتعديل صفحة معينة: عدل ملف HTML في `pages/` وملف JS في `js/pages/`
- لتعديل الوظائف المشتركة: عدل الملفات في `js/core/`
- لتعديل التصميم: عدل `css/styles.css`
